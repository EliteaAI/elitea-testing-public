# Test Coverage Metrics

## Overview

Test coverage measures how much of the application is tested.

## Coverage Types

### Feature Coverage
Percentage of features with test cases.

### Code Coverage
Percentage of code executed by automated tests (where applicable).

### Requirements Coverage
Percentage of requirements with test verification.

## Measuring Coverage

- Track test cases per feature
- Map tests to requirements
- Monitor automation coverage
- Report coverage gaps

## Reporting

Coverage metrics should include:
- Overall coverage percentage
- Coverage by feature area
- Coverage gaps and risks
- Trends over time
