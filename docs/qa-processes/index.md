## Process Principles

1. **Test early and often** — Shift left in the development cycle
2. **Automate where possible** — Free up time for exploratory testing
3. **Document clearly** — Make testing reproducible
4. **Collaborate actively** — Work closely with developers and PMs
5. **Focus on user impact** — Prioritize testing based on user value
