# Team Roles and Responsibilities

## QA Lead

### Responsibilities
- Define QA strategy and processes
- Lead bug triage
- Manage QA team
- Report on quality metrics
- Coordinate with engineering and product
- Own release sign-off

### Key Activities
- Sprint planning participation
- Weekly triage meetings
- Release readiness reviews
- Process improvement initiatives

## QA Engineer

### Responsibilities
- Execute test cases
- Report and verify bugs
- Create and maintain test data
- Participate in test planning
- Provide quality feedback

### Key Activities
- Daily testing activities
- Bug reporting and verification
- Test case creation
- Exploratory testing
- Regression testing

## Automation Engineer

### Responsibilities
- Develop automated tests
- Maintain test frameworks
- Improve test coverage
- Support CI/CD integration

### Key Activities
- Write and maintain automated tests
- Review and refactor test code
- Integrate tests with CI/CD
- Train team on automation tools

## Collaboration

All roles collaborate on:
- Test planning
- Bug triage
- Process improvement
- Knowledge sharing
