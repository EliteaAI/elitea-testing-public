# Onboarding Checklist

## Access and Setup

- [ ] GitHub organization access
- [ ] ELITEA Board access
- [ ] DEV environment access
- [ ] STAGE environment access
- [ ] Test accounts credentials
- [ ] Team communication channels
- [ ] Password manager access
- [ ] Documentation repository cloned

## Knowledge

### Week 1
- [ ] Read QA Handbook overview
- [ ] Read Bug Reporting Guide
- [ ] Read Bug Priority and Severity Guide
- [ ] Read Project Fields Guide
- [ ] Review ELITEA Board structure
- [ ] Understand environment differences

### Week 2
- [ ] Read Test Execution guides
- [ ] Review test automation setup
- [ ] Shadow test execution
- [ ] Read Labels and Taxonomy
- [ ] Review recent bugs

### Month 1
- [ ] Execute tests independently
- [ ] Report bugs following templates
- [ ] Participate in bug triage
- [ ] Contribute to test planning
- [ ] Learn test automation basics

## Activities

- [ ] Attended team meeting
- [ ] Met with QA lead
- [ ] Paired with experienced QA
- [ ] Executed first test case
- [ ] Reported first bug
- [ ] Participated in triage
- [ ] Reviewed test automation

## Feedback

- [ ] Week 1 check-in
- [ ] Week 2 check-in
